ANIMATED BUNNY

VIEW ON GITHUB

Click on the numbers to interact! Scroll down to play the animation

When the assignment to create a multi-button image page, I immediately wanted to make an animation. This project coincided with a video project where I drew the individual frames of this bunny walking so I knew I wanted to use this in the website. The original animation was a 18 frame animation but I felt it was enough to cut the frames into the essential 8 for a walking cycle.
